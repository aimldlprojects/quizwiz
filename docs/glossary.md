# Glossary

Short meanings for the words used in the app docs.

| Term | Meaning |
| --- | --- |
| Active profile | The learner profile currently in use on this device. |
| Active path | The branch of subject and topic chips the user is currently looking at. |
| Selected topics | The topics the user wants to learn or practice. |
| Visible topics | The subjects and topics the Admin screen allows the user to see. |
| Local | Saved on this device first. |
| Global | Saved on the shared database so another device can get it later. |
| Synced | The device and global data match for that item. |
| Pending | A change exists on the device or on the server and still needs sync. |
| Dirty | The device has local changes that still need to be sent. |
| Learn | The card study screen. |
| Practice | The question-answer screen. |
| Admin permissions | The show or hide rules for subjects and topics. |
