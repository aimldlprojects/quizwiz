# Data Ownership

Use this as the quick guide for what stays on the device and what goes to the global database.

| Data | Saved locally right away? | Sent to the global database by sync? | Notes |
| --- | --- | --- | --- |
| Active profile | Yes | No | This is the current learner on the device. |
| Topic selections | Yes | Yes | The user picks these in Topics. |
| Learn progress | Yes | Yes | The last study position for each topic is kept. |
| Practice progress | Yes | Yes | Includes answers, review history, and accuracy. |
| Badges | Yes | Yes | Badge progress follows the active profile. |
| Theme and voice settings | Yes | Yes | These are Profile preferences. |
| Sync mode, sync interval, sync gap | Yes | Yes | These are shared sync settings in Profile. |
| Admin allowed subjects and topics | Yes | Yes | These control what the user can see. |
| Current admin topic path | Yes | Yes | This is also shared across devices. |
| Disable or enable user | Yes | Yes | This affects which learner profiles are shown. |
| Screen expansion state | Yes | No | This is just the current view on the device. |
