# Practice

## Purpose

Answer questions for the selected topic.

## Status

Interactive screen.

## Actions

- Answer
- Show answer
- Mark result
- Next question
- Toggle auto next
- Change delays
- Toggle random order
- Tap sync

## Notes

- Practice progress is saved for the active profile

## Sync Notes

- Practice answers and review progress save locally first
- The sync button sends them to the global database
