# Badges

## Purpose

View progress rewards.

## Main Flow

- The screen starts with `Level Badges`.
- The page shows medal badges.
- The page also shows `Achievement Stickers`.
- Earned badges follow the active profile.

## Controls

- View medal badges - see the main progress medals.
- View unlocked stickers - see the badges the user has earned.

## Notes

- Badges are based on accuracy, streak, mastery, and learning volume.
- If no user is active, the screen asks you to choose a user profile first.
- While loading, the screen shows `Loading badges...`.
