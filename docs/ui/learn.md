# Learn

## Purpose

Study flash cards for the selected topic.

## Main Flow

- The screen starts with `Learn mode`.
- The selected topic name appears below the action icons.
- The topic name stays on one line.
- The main card shows the question first.
- Flipping the card shows the answer side and reads it aloud.
- Moving next or previous changes the current card.
- The `Card X of Y` line shows the full deck count for the selected topic.
- Random order reshuffles the remaining cards when you turn it on.

## Controls

- Study cards - show and read the question for the selected topic.
- Flip cards - show and read the answer for the current card.
- Move next or previous - move through the deck.
- Toggle random order - shuffle the current and remaining cards for the selected topic.
- Toggle autoplay - turn automatic flipping and moving to next card on or off.
- Text to speech - read the front side when the card appears and read the back side after flip.
- Play card - read the side that is currently showing.

## Notes

- Resumes from the last saved card position.
- Random order reshuffles the current deck in place when you turn it on.
- In random order, the card buttons follow the shuffled deck order from the current position.
- Table topics move forward without repeating cards until the deck is finished.
- If no topic is selected, Learn asks you to choose one in Topics.
- When a topic is complete, Learn shows `Topic complete`.
- The completion card fades and slides in, then shows a `Learn more` button to restart the same topic from the beginning.
- `Learn more` has a little extra internal padding so the text stays clear of the edge.
- `Learn more` uses the same restart button style as Practice.

## Sync Notes

- Learn progress, including the saved card position for the same topic, saves locally first.
- The top-right sync icon or the Profile screen sync controls send that Learn progress to the global database.
- Learn progress updates from card actions and from the next focus cycle, not from sync completion alone.
- `Learn more` keeps history but restarts the same topic session from the first card.
